# Demoblaze Robot Framework Automation

## Project Description

This is a professional test automation project for the [Demoblaze](https://www.demoblaze.com/) e-commerce website using Robot Framework and Selenium WebDriver. The project follows the Page Object Model (POM) design pattern and includes comprehensive test coverage for:

- Home page functionality
- Product categories (Phones, Laptops, Monitors)
- Product details and interactions
- Shopping cart operations
- Order placement
